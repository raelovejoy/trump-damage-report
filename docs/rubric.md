# Rubric

## Labels
- Confirmed: supported by reliable sources (or primary document)
- Contested: sources disagree or only partisan/unverified claims exist

## Confidence (0–100)
- 90–100: primary document + multiple reputable confirmations
- 70–89: multiple reputable confirmations
- 50–69: single reputable source or partial documentation
- <50: contested / developing / unclear

## Severity (0–100)
Consider:
- scope (how many people/institutions affected)
- irreversibility (can it be undone)
- rights impact (civil/human rights, safety)
- institutional capture risk
- time horizon (immediate vs long-tail)
